# 跨语种适配器 / Cross-Language Adapters

> 本目录包含针对不同目标语种使用本协议时的参数调整指南。

---

## 适配原理

法语AI学习协议的核心——**工具分层、提问类型化、错误分类、验证机制**——具有跨语种可迁移性。但具体到不同语种，需要做以下三个层面的调整：

1. **可信度矩阵的数值调整**（[02-credibility-matrix.md](../protocol/02-credibility-matrix.md)）
2. **错误类型实例的本地化**（[03-error-taxonomy.md](../protocol/03-error-taxonomy.md)）
3. **提问模板中的术语替换**（[01-prompt-templates.md](../protocol/01-prompt-templates.md)）

## 如何选择或创建适配器

### 选择已有适配器

如果你正在学习的语言已经有适配器，直接使用即可。

### 创建新适配器

如果没有你的目标语种适配器，可以基于 [`template-blank.md`](./template-blank.md) 创建一个。创建时需要：

1. **调研该语种的学习特点**（与法语相比的难度点、常见错误类型）
2. **调整可信度矩阵**（基于该语种 AI 训练数据的稀缺程度）
3. **本地化错误类型实例**（用该语种的真实案例替换法语案例）
4. **替换提问模板中的术语**（如"虚拟式"换成该语种对应的语法概念）
5. **测试与迭代**（用新适配器进行 1-2 周的实际使用后调整）

## 适配器难度参考

不同语种适配的难度不同，主要取决于两个因素：

1. **与法语的形态相似度**：形态越相似，需要调整的越少
2. **AI 训练数据中该语种的占比**：占比越低，AI 整体表现越差，需要更严格的验证

| 语种 | 与法语的形态相似度 | AI训练数据占比 | 适配难度 |
|---|---|---|---|
| 西班牙语 | 高（同属罗曼语族） | 中等 | ★★☆☆☆ 简单 |
| 意大利语 | 高 | 较低 | ★★☆☆☆ 简单 |
| 葡萄牙语 | 高 | 较低 | ★★★☆☆ 中等 |
| 德语 | 中 | 中等 | ★★★☆☆ 中等 |
| 英语 | 中（共享大量词汇） | 极高 | ★★☆☆☆ 简单 |
| 日语 | 低（完全不同的书写系统） | 中等 | ★★★★☆ 较难 |
| 阿拉伯语 | 低（不同的书写和形态） | 较低 | ★★★★☆ 较难 |
| 汉语 | 极低 | 极高 | ★★★☆☆ 中等 |
