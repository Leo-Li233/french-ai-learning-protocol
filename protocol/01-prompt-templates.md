# 五类提问模板 / Five Prompt Templates

> 适用场景：任何 AI 辅助语言学习的场景
> 核心理念：提问质量决定 AI 回答质量

---

## 1. 验证型 / Verification

**适用场景：** 写作即时纠错、单句翻译确认、变位验证

**模板：**
```
我写的这个句子对吗？[你的句子]
如果错了，错在哪里？请给出正确版本和简短的规则说明。
```

**进阶变体：**
```
我翻译的这句话：'[中文]'
AI 翻译结果：'[AI的翻译]'
请对照检查：是否准确？有没有更地道的表达？
```

**示例：**
> 我写的这个句子对吗？Je suis allé au cinéma avec mon amis. 如果错了，错在哪里？

---

## 2. 归纳型 / Synthesis

**适用场景：** 零碎知识系统化、相近概念对比、知识点整理

**模板：**
```
请总结 [概念A] 和 [概念B] 的核心区别，包括：
1. 定义对比
2. 典型使用场景（各给三个例句）
3. 常见混淆点和避免方法
4. 考试/实际使用中需要注意的陷阱
```

**进阶变体：**
```
请帮我整理 [主题] 的完整知识体系，用结构化形式呈现：
- 核心分类
- 各类别的定义和使用规则
- 易错点
- 配套练习题（5道）
```

**示例：**
> 请总结复合过去时和未完成过去时的核心区别，包括定义对比、典型使用场景、常见混淆点和考试陷阱。

---

## 3. 生成型 / Generation

**适用场景：** 错题转训练题、扩展练习、按难度分级练习

**模板：**
```
根据我上面的 [数量] 道错题，请生成 [数量] 道同类型的训练题。
要求：
1. 难度与原题相当
2. 涵盖相同的知识点
3. 提供参考答案和简短的解析
```

**进阶变体：**
```
请为 [语法规格/词汇主题] 生成 [数量] 道分级练习题：
- 基础 [N] 道
- 中级 [N] 道
- 高级 [N] 道
每道题附答案和解析。
```

**示例：**
> 根据我上面这二十道错题，请生成五道同类型的法语虚拟式练习题，难度相当，提供参考答案和解析。

---

## 4. 预测型 / Prediction

**适用场景：** 备考前瞻规划、考试重点分析、考点预判

**模板：**
```
以下是 [考试/测验] 的大纲/范围：
[粘贴大纲内容]
请基于此：
1. 预测最可能考的 [数量] 个高频考点
2. 每个考点给出可能的出题形式
3. 给出每个考点的备考建议
4. 列出我的知识盲区（基于我的背景：[描述你的情况]）
```

**进阶变体：**
```
我即将参加 [考试名称]，请扮演出题老师，根据近 [N] 年的命题趋势：
1. 预测今年的出题方向
2. 为每个方向出一道模拟题
3. 提供评分标准
```

**示例：**
> 以下是今年法语专四考试大纲，请预测最可能考的三种听写文章主题，并给出每个主题的常见词汇和句型。

---

## 5. 反思型 / Reflection

**适用场景：** 错题模式诊断、知识盲区识别、学习计划调整

**模板：**
```
我最近 [时间段] 的错题集中在 [主题/类型]。请：
1. 分析可能存在的知识盲区
2. 推断背后的认知误区
3. 给出针对性的补漏计划（包括学习资源、练习建议、时间安排）
4. 设计3个自检问题，帮助我判断自己是否真正理解
```

**进阶变体：**
```
这是我最近 [数量] 次练习的成绩：[粘贴数据]
请：
1. 识别成绩波动的模式
2. 分析波动背后的可能原因
3. 给出未来 [时间段] 的学习策略调整建议
```

**示例：**
> 我最近五天的错题集中在虚拟式，请分析可能的知识盲区，给出补漏计划，并设计3个自检问题。

## 模板使用进阶指南

### 从初级到高级的提问路径

```
阶段1（初期）   → 验证型为主，问题简短
阶段2（中期）   → 引入归纳型，开始要求结构化输出
阶段3（中后期） → 加入生成型，错题转化为训练
阶段4（备考期） → 预测型成为主流
阶段5（沉淀期） → 反思型最频繁，元认知主导
```

### 模板组合策略

复杂学习任务往往需要组合多个模板：

- **写作提升**：验证型（初稿）→ 反思型（问题诊断）→ 归纳型（规则学习）→ 生成型（练习巩固）
- **考前冲刺**：预测型（考点预判）→ 归纳型（重点知识）→ 生成型（模拟题）→ 反思型（错题复盘）
- **新知识点**：归纳型（建立框架）→ 生成型（基础练习）→ 验证型（应用检验）→ 反思型（盲区识别）

### 提问的元原则

1. **具体 > 抽象**：不要问"帮我学法语"，而要问"帮我分析这句法语为什么错了"
2. **任务导向 > 知识导向**：不要问"虚拟式是什么"，而要问"我这句话该不该用虚拟式"
3. **可验证 > 不可验证**：问完之后知道去哪里查证答案的问题更有效
4. **自我反思 > 工具依赖**：每三四个模板中至少包含一个反思型
